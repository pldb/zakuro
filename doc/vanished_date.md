# 滅日

没日を端的に説明すると、1朔望月（29.5..日）を30日で表わすために、一定の法則で `不足した日` として配置された日である

経朔（平均値で求めた月初日）をもとに計算を行う

## 計算
「長慶宣明暦算法」に従う

下記は該当箇所の引用と、それに対応する計算を示している

```
◯滅日ヲ求ルノ術
```
